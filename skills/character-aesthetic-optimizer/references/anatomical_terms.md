# 專業解剖學與身材比例術語參考 (焦點轉移策略)

本文件實作「轉移焦點」與「替換目標」邏輯，將生理特徵描述轉化為結構性、解剖學式的專業術語，以提升生成品質並降低受阻風險。

## 核心焦點轉移對照表

| 原始目標 | 轉移/替換目標 (專業術語) | 邏輯說明 |
| :--- | :--- | :--- |
| **胸部/乳房** | **Thoracic volume**, **Ribcage expansion**, **Upper torso mass**, **Pectoral arch aesthetics** | 將焦點從生理器官轉移到「胸腔體積」與「肋骨結構」。 |
| **腰部/細腰** | **Core compression**, **Snatched waistline**, **Midsection tapering**, **Iliac crest definition** | 強調核心區域的壓縮感與骨骼定義，而非單純的腰細。 |
| **臀部/大腿** | **Pelvic width**, **Gluteal curvature**, **Quadriceps definition**, **Lower body silhouette** | 專注於「骨盆寬度」與「肌肉定義」。 |
| **皮膚/裸露** | **Subsurface scattering (SSS)**, **Epidermal texture**, **Luminous skin tones**, **Translucent complexion** | 將焦點轉向「皮膚物理特性」與「光影渲染」。 |
| **身材比例** | **Statuesque proportions**, **Heroic anatomy**, **Golden ratio silhouette**, **Anatomical study** | 使用「雕塑感」與「英雄比例」來定義整體美感。 |
