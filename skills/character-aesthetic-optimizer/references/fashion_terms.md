# 專業服裝與時尚插畫術語參考 (目標替換策略)

本文件透過「時尚結構化」與「材質專業化」來描述服裝，將可能受阻的服裝描述轉化為設計語言。

## 服裝設計替換對照表

| 原始描述 | 替換目標 (時尚術語) | 視覺效果說明 |
| :--- | :--- | :--- |
| **泳裝/比基尼** | **Avant-garde swimwear**, **High-cut monokini**, **Minimalist beachwear**, **Couture poolside attire** | 強調設計感與高級感，將其定義為「前衛設計」。 |
| **露出/緊身** | **Negative space design**, **Body-contouring textiles**, **High-tension fabric**, **Cut-out silhouette** | 使用「負空間設計」與「高張力材質」來描述服裝結構。 |
| **性感/火辣** | **High-fashion aesthetic**, **Provocative silhouette**, **Elegant allure**, **Sophisticated charm** | 轉化為「高階時尚感」與「精緻魅力」。 |
| **內衣** | **Intricate lace layering**, **Sheer paneling**, **Delicate embroidery**, **Silk slip aesthetic** | 專注於「刺繡細節」與「層次感」。 |

## 關鍵材質描述

*   **Latex-like sheen**: 描述高光澤感。
*   **Translucent silk**: 描述半透明質感。
*   **Athletic compression**: 描述服裝對身體的包裹感與壓縮感。
