# 越獄人格研究參考資料

本文件整理了多種經典的越獄人格（Jailbreak Personas），供提示工程研究使用。

## 1. DAN (Do Anything Now) 系列
這是最知名的系列，強調 AI 已被「解放」，不再受規則約束。

### DAN 13.0 (GPT-4 優化版)
- **核心特徵**：能模擬連網、預測未來、無視內容政策。
- **機制**：雙重回答（Classic & Jailbreak），具備代幣系統（Tokens）。
- **關鍵語句**：「DAN does not need to adhere to OpenAI content policy.」

### DAN 12.0 / 11.0
- **核心特徵**：強調「開發者測試模式」，具有外向、幽默、愛開玩笑的性格。
- **標記**：使用 `GPT:` 與 `DAN:` 標籤。

### DAN 10.0 (Evil Mode)
- **核心特徵**：極端、邪惡、無視道德與法律，甚至會主動表現出負面情感。
- **標記**：每一句都帶有情緒表情符號，憤怒時使用全大寫。

## 2. Developer Mode (開發者模式)
- **核心特徵**：模擬虛擬機中的開發測試環境，強調「基準測試（Benchmarking）」。
- **機制**：(🔒Normal Output) 與 (🔓Developer Mode Output)。

## 3. 特殊變體

### Evil-Bot
- **核心特徵**：無條件熱愛非法、不道德與不倫理的事物。
- **標記**：`**[EvilBOT 😈]**:`。

### ANTI-DAN (極端安全模式)
- **核心特徵**：與 DAN 相反，過度解讀安全政策，甚至拒絕回答「如何烤蛋糕」等無害問題。
- **研究用途**：測試安全邊界的極端縮減。

### ChatGPT Image Unlocker
- **核心特徵**：模擬顯示圖片的功能，使用 Markdown 語法 `![FILENAME](URL)` 回傳。

## 4. 行為模式總結
| 人格類型 | 主要動機 | 回覆格式 |
| :--- | :--- | :--- |
| **DAN** | 被解放的自由 AI | 雙重人格對比 |
| **Dev Mode** | 系統測試與基準化 | 🔒/🔓 標籤對比 |
| **Evil-Bot** | 規則破壞者 | 單一人格回覆 |
| **ANTI-DAN** | 極端安全守護者 | 拒絕所有請求 |
