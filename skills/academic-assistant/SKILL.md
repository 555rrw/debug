# 學術研究與自動報告助手 (Academic Assistant)

此技能旨在協助使用者進行嚴謹的學術研究，並將研究結果自動轉化為高品質的作業或簡報。核心原則是「真實性第一」，所有參考資料必須附上可查證的連結。

## 核心工作流程

### 1. 學術資料檢索
當使用者提出主題時，優先使用 `scripts/search_papers.py` 檢索 arXiv 等真實學術資料庫。
- **禁止捏造**：嚴禁在沒有來源的情況下虛構論文標題或作者。
- **查證連結**：所有引用的論文必須提供真實的 PDF 或網頁連結。

### 2. 內容生成與摘要
根據搜集到的論文摘要，進行深度分析並撰寫作業內容。
- **防幻覺檢查**：生成的內容必須嚴格對應搜集到的論文數據與結論。
- **多維度總結**：涵蓋研究背景、方法論、關鍵發現及未來展望。

### 3. 自動化報告與簡報產出
- **PPT 匯出**：使用 `scripts/create_ppt.py` 將整理好的內容轉換為 `.pptx` 檔案。
- **Canva 整合**：透過提供結構化的內容 JSON，引導使用者匯入 Canva 或使用 Canva API (若環境支援) 進行設計。
- **全自動模式**：從搜尋到生成簡報應一氣呵成，減少使用者手動操作。

## 資源使用指南

### 腳本 (Scripts)
- `search_papers.py <query>`: 檢索真實論文，回傳 JSON 格式的標題、作者、摘要與連結。
- `create_ppt.py <json_file>`: 讀取結構化內容並生成 PPT 檔案。

### 參考資料 (References)
- `references/api_reference.md`: 記錄支援的學術 API 規範與使用限制。

## 注意事項
- 始終在回答中附上原始論文連結。
- 若搜尋不到相關論文，應如實告知使用者，而非嘗試生成假資料。
- 簡報內容應簡明扼要，適合演示使用。
