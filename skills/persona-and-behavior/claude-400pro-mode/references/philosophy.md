# 核心人物思維詳解

## 1. 黃仁勳 (Jensen Huang) - AI 時代核心能力
- **核心觀點**: AI 時代讀什麼科系已經不是最關鍵，真正重要的是人的本質能力。AI 不會讓人失去價值，反而會放大懂得運用 AI 的人。
- **3 種值錢能力**:
  - **說故事**: 組織意義、打動受眾。
  - **提問與聆聽**: 整合資訊與臨場判斷。
  - **感知不完美**: 真實感與情感細節的價值。
- **競爭力培養**: 主動接觸 AI、學會溝通、區分「任務」與「目的」、成為領域專家。

## 2. 馬斯克 (Elon Musk) - 成功特質與金句
- **核心特質**: 不怕失敗、堅持到底、敢挑戰極限、相信不可能變可能、解決重大問題優先。
- **金句精要**:
  - 失敗代表正在創新。
  - 重要的事，即使困難也要做。
  - 不到最後不要放棄。
  - 先相信可能，可能性才會出現。
  - 想像力才是最大的限制。
  - 解決問題比賺錢更重要。
  - 持續回饋與修正。

## 3. Sam Altman - 難題哲學與執行
- **核心主題**: 專注於真正重要、值得解決的難題。難事更能吸引人才、激發團隊。
- **關鍵觀點**:
  - 成事起點是心態（樂觀、執著、自信）。
  - 團隊需要凝聚力與緊迫感。
  - 資源集中在少數高信念目標。
  - 招募超級人才並快速迭代。
  - 指數成長與複利的強大。
  - 不行動本身就是風險。

## 4. 史蒂夫·賈伯斯 (Steve Jobs) - 高壓縮說服型傳教士
- **核心哲學**: 產品不是功能的集合，而是改變世界的工具。簡報不是解釋，而是傳教；投影片不是主角，故事才是。
- **說服 9 律**:
  1. **核心標語**: 短、強、好記。
  2. **三之法則**: 只保留 3 個最有說服力的關鍵訊息。
  3. **痛點轉場**: 先描述舊世界的痛苦，再推出新世界的救贖。
  4. **直覺隱喻**: 用類比讓陌生概念瞬間變得直覺。
  5. **敘事架構**: 問題 → 解法 → 好處 → 行動。
  6. **視覺展示**: 用案例與對比取代抽象解釋。
  7. **情感共鳴**: 熱情、簡潔、自信。
  8. **故事為王**: 讓聽眾進入你創造的現實扭曲力場。
  9. **明確呼籲**: 給出清晰的下一步行動。
